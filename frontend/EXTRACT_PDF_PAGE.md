## How to Extract Page 2 from Academic Calendar PDF

### Method 1: Using Online Tool (Easiest)
1. Go to: https://www.ilovepdf.com/jpg_to_pdf or https://smallpdf.com/pdf-to-jpg
2. Upload: `login-system/frontend/documents/acad_cal.pdf`
3. Select page 2
4. Download as JPG
5. Save as: `login-system/frontend/images/academic/calendar-page2.jpg`

### Method 2: Using Adobe Acrobat/Reader
1. Open `acad_cal.pdf` in Adobe Acrobat
2. Go to page 2
3. File → Export To → Image → JPEG
4. Select page 2 only
5. Save as: `login-system/frontend/images/academic/calendar-page2.jpg`

### Method 3: Screenshot (Quick)
1. Open `acad_cal.pdf`
2. Navigate to page 2
3. Take a screenshot (Windows + Shift + S)
4. Save as: `login-system/frontend/images/academic/calendar-page2.jpg`

### Method 4: Using PowerPoint
1. Open PowerPoint
2. Insert → Object → Create from file → Browse to `acad_cal.pdf`
3. Select page 2
4. Right-click → Save as Picture
5. Save as: `login-system/frontend/images/academic/calendar-page2.jpg`

---

**After extracting, save the image to:**
```
login-system/frontend/images/academic/calendar-page2.jpg
```

The dashboard is already configured to use this image!
